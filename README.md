## 💡 Getting Started

### Prerequisites

```
$ node -v
v12.16.1

$ npm -v
v6.13.4
```

### Installation
1. `clone` the repository,
```
$ git clone "https://github.com/do-mi-sol/domisol-web-application.git"
```

2. `Install` dependencies,
```
$ npm install
```       
3. `start` the project,
```
$ npm start
```
