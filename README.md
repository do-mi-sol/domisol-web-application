## 💡 Features
* JS기반 react 프레임워크 사용

* 도움이 필요한 사용자는 도미솔 게시판에 SNS 캡쳐 사진과 함께 게시글 업로드

* 댓글과 좋아요를 통해 작성자에게 피드백

* 한 눈에 보기 쉬운 UI 구성

## 💡 Getting Started

### Prerequisites

```
$ node -v
v12.16.1

$ npm -v
v6.13.4
```

### Installation
1. `clone` the repository,
```
$ git clone "https://github.com/do-mi-sol/domisol-web-application.git"
```

2. `Install` dependencies,
```
$ npm install
```       
3. `start` the project,
```
$ npm start
```
